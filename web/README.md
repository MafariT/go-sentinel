# Go-Sentinel Frontend

The dashboard for Go-Sentinel, a lightweight service monitoring tool built with React, TypeScript, and Tailwind CSS.

## Features
- Real-time monitor status
- Latency trend visualization
- Easy monitor management

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
The build output will be located in `dist/`, which is embedded into the Go binary.
